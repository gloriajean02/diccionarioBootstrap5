Alertas en Bootstrap 5

Alerta Success

La clase .alert-success se usa para mostrar un mensaje de éxito con un fondo verde.

Alerta success
<div class="alert alert-success">Alerta success</div>

Alerta Primary

La clase .alert-primary se utiliza para mostrar una alerta con un fondo azul, comúnmente usada para información importante.

Alerta primary
<div class="alert alert-primary">Alerta primary</div>

Alerta Danger

La clase .alert-danger se utiliza para mostrar una alerta con un fondo rojo, comúnmente usada para mensajes de advertencia o error.

Alerta danger
<div class="alert alert-danger">Alerta danger</div>

Alerta con Título y Cuerpo

En este ejemplo, la alerta contiene un encabezado <h4> y un cuerpo de texto en un párrafo.

Título

Este es el cuerpo de la alerta con un mensaje más detallado.

<div class="alert alert-danger">
    <h4 class="alert-heading">Título</h4>
    <p>Este es el cuerpo de la alerta con un mensaje más detallado.</p>
    </div>

Alerta con Botón de Cierre

El botón de cierre se añade con la clase .btn-close y el atributo data-bs-dismiss='alert'. Al hacer clic en el botón, la alerta se cierra.

Título

Este es un cuerpo de texto.

<div class="alert alert-danger" id="myAlert">
    <h4 class="alert-heading">Título</h4>
    <p>Este es un cuerpo de texto.</p>
    <button class="btn-close" data-bs-dismiss="alert"></button>
    </div>

Interacción con JavaScript

En este ejemplo, vamos a utilizar JavaScript para manejar la interacción con las alertas. Usamos el evento close.bs.alert para detectar cuando una alerta se ha cerrado y ejecutar una acción.

Escuchar el Cierre de la Alerta

Para detectar el cierre de la alerta, primero seleccionamos el elemento con document.getElementById y luego añadimos un "listener" de evento con addEventListener.


    // Seleccionamos la alerta con el id 'myAlert'
    const myAlert = document.getElementById("myAlert");

    // Añadimos un listener para detectar el evento 'close.bs.alert'
    myAlert.addEventListener('close.bs.alert', event => {
        console.log('La alerta se ha cerrado');
    });

Este código detecta cuando el evento de cierre de la alerta se activa (es decir, cuando el usuario hace clic en el botón de cerrar o se cierra por otro método), y se ejecuta la función que muestra en la consola el mensaje 'La alerta se ha cerrado'.

Cerrar la Alerta Programáticamente

También podemos cerrar una alerta de forma programática utilizando el método close() de la instancia de la alerta.


    // Creamos una instancia de la alerta
    const myAlertInstance = bootstrap.alert.getOrCreateInstance('#myAlert');

    // Cerramos la alerta programáticamente
    myAlertInstance.close();

El código anterior crea una instancia de la alerta utilizando el método bootstrap.alert.getOrCreateInstance. Luego, utilizamos el método close() para cerrar la alerta automáticamente, sin necesidad de interacción del usuario.
