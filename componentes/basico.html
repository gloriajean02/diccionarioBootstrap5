Componentes básicos en Bootstrap 5

Los botones de Bootstrap tienen diferentes clases para variar su estilo y color. A continuación, puedes ver varios ejemplos con distintos estilos de botones.



<button class="btn btn-primary">Botón primary</button>
<button class="btn btn-secondary">Botón secondary</button>
<button class="btn btn-success">Botón success</button>
<button class="btn btn-danger">Botón danger</button>
<button class="btn btn-warning">Botón warning</button>
<button class="btn btn-info">Botón info</button>
<button class="btn btn-light">Botón light</button>
<button class="btn btn-dark">Botón dark</button>
<button class="btn btn-link">Botón link</button>


Outline

Los botones con clase btn-outline-* tienen un borde, pero sin un fondo colorido. Son útiles para combinar con fondos de colores más suaves o para dar un toque sutil a los botones.

Botón dark 

<button class="btn btn-outline-info">Botón info</button>
<button class="btn btn-outline-success">Botón success</button>
<a href="#" class="btn btn-outline-dark">Botón dark</a>


Tamaños

Los botones pueden tener distintos tamaños para ajustarse mejor a la interfaz de usuario. Puedes usar clases como btn-sm para botones pequeños, btn-md para botones medianos y btn-lg para botones grandes.

Botón large 

<button class="btn btn-outline-info btn-sm">Botón small</button>
<button class="btn btn-outline-success btn-md">Botón middle</button>
<a href="#" class="btn btn-outline-dark btn-lg">Botón large</a>


Botones agrupados

Los botones también pueden agruparse dentro de un div con la clase btn-group para formar un conjunto de botones alineados horizontalmente.



<div class="btn-group">
    <button class="btn btn-primary">Botón primary</button>
    <button class="btn btn-secondary">Botón secondary</button>
    <button class="btn btn-success">Botón success</button>
</div>


Botones check

Los botones de tipo "check" se utilizan para representar acciones de selección, como un "toggle". Se pueden emplear con etiquetas de input del tipo checkbox.



<div class="btn-group">
    <input type="checkbox" class="btn-check" id="btncheck1">
    <label class="btn btn-outline-secondary" for="btncheck1">Botón secondary</label>
</div>


Badge

Las badge se utilizan para mostrar información adicional, como un número de mensajes o notificaciones.



<button class="btn btn-primary">
        Message <span class="badge text-bg-secondary rounded-pill">3</span>
    </button>
