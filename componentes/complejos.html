Componentes complejos en Bootstrap 5

Nav

La clase nav se utiliza para crear una barra de navegación básica, donde se pueden agregar enlaces. Este componente es comúnmente usado para crear menús.

  • Opción 1
  • Opción 2
  • Opción 3
  • Dropdown
    • Item 1
    • Item 2
    • Item 3

<ul class="nav">
        <li class="nav-item">
            <a class="nav-link" href="#">Opción 1</a>
        </li>
        <li class="nav-item">
            <a class="nav-link" href="#">Opción 2</a>
        </li>
        <li class="nav-item">
            <a class="nav-link disabled" href="#">Opción 3</a>
        </li>
        <li class="nav-item">
            <a class="nav-link dropdown-toggle" data-bs-toggle="dropdown">Dropdown</a>
            <ul class="dropdown-menu">
                <li><a href="#" class="dropdown-item">Item 1</a></li>
                <li><a href="#" class="dropdown-item">Item 2</a></li>
                <li><hr class="dropdown-divider"/></li>
                <li><a href="#" class="dropdown-item">Item 3</a></li>
            </ul>
        </li>
    </ul>


Justificado en el centro

Para centrar los elementos dentro de la barra de navegación, utilizamos la clase justify-content-center que hace que los enlaces se distribuyan en el centro.

  • Opción 1
  • Opción 2
  • Opción 3

<ul class="nav justify-content-center">
        <li class="nav-item"><a class="nav-link" href="#">Opción 1</a></li>
        <li class="nav-item"><a class="nav-link" href="#">Opción 2</a></li>
        <li class="nav-item"><a class="nav-link disabled" href="#">Opción 3</a></li>
    </ul>

Justificado en el inicio

La clase justify-content-start permite alinear los elementos de la barra de navegación hacia el inicio (izquierda) de la página.

  • Opción 1
  • Opción 2
  • Opción 3

<ul class="nav justify-content-start">
        <li class="nav-item"><a class="nav-link" href="#">Opción 1</a></li>
        <li class="nav-item"><a class="nav-link" href="#">Opción 2</a></li>
        <li class="nav-item"><a class="nav-link disabled" href="#">Opción 3</a></li>
    </ul>

Justificado en el final

Con la clase justify-content-end, los elementos de la barra de navegación se alinean al final (derecha) de la página.

  • Opción 1
  • Opción 2
  • Opción 3

<ul class="nav justify-content-end">
        <li class="nav-item"><a class="nav-link" href="#">Opción 1</a></li>
        <li class="nav-item"><a class="nav-link" href="#">Opción 2</a></li>
        <li class="nav-item"><a class="nav-link disabled" href="#">Opción 3</a></li>
    </ul>

Alineado en columnas

Si deseas que los enlaces se muestren en una columna, puedes usar la clase flex-column para transformar el menú en una lista vertical.

  • Opción 1
  • Opción 2
  • Opción 3

<ul class="nav flex-column">
        <li class="nav-item"><a class="nav-link" href="#">Opción 1</a></li>
        <li class="nav-item"><a class="nav-link" href="#">Opción 2</a></li>
        <li class="nav-item"><a class="nav-link disabled" href="#">Opción 3</a></li>
    </ul>

Nav en tipo pills

La clase nav-pills crea un menú de pestañas (pills), que es un diseño con pestañas redondeadas y activadas.

  • Opción 1
  • Opción 2
  • Opción 3

<ul class="nav nav-pills">
        <li class="nav-item"><a class="nav-link" href="#">Opción 1</a></li>
        <li class="nav-item"><a class="nav-link active" href="#">Opción 2</a></li>
        <li class="nav-item"><a class="nav-link disabled" href="#">Opción 3</a></li>
    </ul>
