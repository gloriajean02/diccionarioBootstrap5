Dropdown en Bootstrap 5


Este es un ejemplo de cómo implementar un menú desplegable (dropdown) utilizando el componente dropdown de Bootstrap. El botón es el que activa el menú, y este contiene elementos que pueden ser seleccionados. Puedes ver comentarios en el código explicando el js.



Visualización del código

Dropdown


<button class="btn btn-success dropdown-toggle" data-bs-toggle="dropdown">
    Mi listado
    <ul class="dropdown-menu">
        <li>
            <a href="#" class="dropdown-item">Item 1</a>
        </li>
        <li>
            <a href="#" class="dropdown-item">Item 2</a>
        </li>
        <li>
            <hr class="dropdown-divider"/>
        </li>
        <li>
            <a href="#" class="dropdown-item">Item 3</a>
        </li>
    </ul>
</button>

JavaScript


// 1. Obtención del elemento del dropdown mediante su ID
    const myDropdown = document.getElementById("myDropdown");
// 2. Añadir un event listener para escuchar cuando el dropdown se despliegue
    myDropdown.addEventListener('show.bs.dropdown', event => {
        console.log('show'); // Aquí se muestra un mensaje en la consola cuando el dropdown se despliega
    });
// 3. Añadir un event listener para escuchar cuando el dropdown se oculte
    myDropdown.addEventListener('hide.bs.dropdown', event => {
        console.log('hide'); // Aquí se muestra un mensaje en la consola cuando el dropdown se oculta
    });
// 4. Crear una instancia de la clase Bootstrap para el dropdown
    const myDropdownInstance = bootstrap.Dropdown.getOrCreateInstance('#myDropdown');
// 5. Mostrar el dropdown de manera programática
    myDropdownInstance.show(); // Esto despliega el menú del dropdown
// 6. Ocultar el dropdown de manera programática
    myDropdownInstance.hide(); // Esto oculta el menú del dropdown
