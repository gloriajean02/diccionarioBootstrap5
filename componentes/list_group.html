List group en Bootstrap 5


Este es un ejemplo de cómo utilizar el componente list-group de Bootstrap para crear una lista interactiva. Los elementos de la lista pueden tener diferentes estados, como activo o deshabilitado. También se incluyen ejemplos de cómo manejar eventos mediante JavaScript.

  • Primero
  • Segundo
  • Tercero


Visualización del código

List-group


<ul class="list-group" id="myTab">
        <li class="list-group-item active">
            Primero
        </li>
        <li class="list-group-item">
            Segundo
        </li>
        <li class="list-group-item disabled">
            Tercero
        </li>
    </ul>

JavaScript


// 1. Obtención del elemento de la lista (tab)
            const myTab = document.getElementById("myTab");
// 2. Añadir un event listener para escuchar cuando el tab sea mostrado
            myTab.addEventListener('show.bs.tab', event => {
                console.log('show'); // Aquí se muestra un mensaje en la consola cuando el tab es mostrado
            });
// 3. Añadir un event listener para escuchar cuando el tab sea ocultado
            myTab.addEventListener('hide.bs.tab', event => {
                console.log('hide'); // Aquí se muestra un mensaje en la consola cuando el tab es ocultado
            });
// 4. Añadir un event listener para escuchar cuando el tab esté completamente oculto
            myTab.addEventListener('hidden.bs.tab', event => {
                console.log('hidden'); // Aquí se muestra un mensaje en la consola cuando el tab es completamente ocultado
            });
// 5. Crear una instancia de Bootstrap para el tab
            const myTabInstance = bootstrap.Tab.getOrCreateInstance('#myTab');
// 6. Acceder a un item del tab y mostrarlo programáticamente
            const myTabItem = bootstrap.Tab.getOrCreateInstance(myTabInstance._getChildren()[1]); // Accede al segundo elemento
            myTabItem.show(); // Muestra este item programáticamente
