Modal en Bootstrap 5


Este es un ejemplo de cómo utilizar el componente modal de Bootstrap 5. Los modales permiten mostrar contenido adicional en una ventana emergente, y podemos manejar su apertura y cierre mediante JavaScript.

Visualización del código

HTML del Modal


<button class="btn btn-primary" data-bs-toggle="modal" data-bs-target="#exampleModal">
    Abrir Modal
</button>

<div class="modal fade" id="exampleModal">
    <div class="modal-dialog">
        <div class="modal-content">
            <div class="modal-header">
                <h5 class="modal-title">Modal Título</h5>
                <button type="button" class="btn-close" data-bs-dismiss="modal"></button>
            </div>
            <div class="modal-body">
                Contenido del modal
            </div>
            <div class="modal-footer">
                <button type="button" class="btn btn-secondary" data-bs-dismiss="modal">Cerrar</button>
                <button type="button" class="btn btn-success">Guardar</button>
            </div>
        </div>
    </div>
</div>

JavaScript


// 1. Obtención del elemento del Modal
            const myExampleModal = document.getElementById("exampleModal");
// 2. Añadir un event listener para escuchar cuando el modal se muestre
            myExampleModal.addEventListener('show.bs.modal', event => {
                console.log('Modal mostrado'); // Mensaje en consola cuando el modal se muestra
            });
// 3. Añadir un event listener para escuchar cuando el modal se oculte
            myExampleModal.addEventListener('hide.bs.modal', event => {
                console.log('Modal ocultado'); // Mensaje en consola cuando el modal se oculta
            });
// 4. Crear una instancia del Modal y mostrarlo
            const myExampleModalInstance = bootstrap.Modal.getOrCreateInstance(myExampleModal);
            myExampleModalInstance.show();
