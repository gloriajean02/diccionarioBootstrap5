Navbar en Bootstrap 5


Este es un ejemplo de cómo utilizar el componente navbar de Bootstrap 5. Un navbar se usa para crear barras de navegación y puede incluir elementos como enlaces, formularios, botones y menús desplegables.

Visualización del código

HTML del Navbar


<div class="navbar navbar-expand-lg bg-body-tertiary"> <!-- Barra de navegación con fondo terciario que se expande en pantallas grandes -->
        <div class="container-fluid"> <!-- Contenedor fluido para la barra de navegación, asegura que ocupe todo el ancho disponible -->
            <a href="#" class="navbar-brand">Marca</a> <!-- Enlace que actúa como marca o logo en la barra de navegación -->
    
            <button class="navbar-toggler" type="button" data-bs-toggle="collapse" data-bs-target="#collapseNavbarContent">
                <span class="navbar-toggler-icon"></span> <!-- Botón para alternar la visibilidad del menú en pantallas pequeñas -->
            </button>
    
            <div class="collapse navbar-collapse" id="collapseNavbarContent"> <!-- Contenedor que contiene el menú de navegación, se oculta en pantallas pequeñas y se muestra al activar el botón -->
                <ul class="navbar-nav me-auto"> <!-- Lista de elementos de la barra de navegación alineados a la izquierda -->
                    <li class="nav-item"> <!-- Elemento de lista de navegación para "Home" -->
                        <a href="#" class="nav-link">Home</a> <!-- Enlace a la página principal -->
                    </li>
                    <li class="nav-item"> <!-- Elemento de lista de navegación para "Services" -->
                        <a href="#" class="nav-link">Services</a> <!-- Enlace a la sección de servicios -->
                    </li>
                    <li class="nav-item"> <!-- Elemento de lista de navegación para "About us" -->
                        <a href="#" class="nav-link">About us</a> <!-- Enlace a la sección sobre la empresa -->
                    </li>
                    <li class="nav-item dropdown"> <!-- Elemento de lista para un menú desplegable -->
                        <a href="#" class="nav-link dropdown-toggle" data-bs-toggle="dropdown">Dropdown</a> <!-- Enlace que activa el menú desplegable -->
                        <ul class="dropdown-menu"> <!-- Menú desplegable con opciones -->
                            <li><a href="#" class="dropdown-item">Opción 1</a></li> <!-- Primer elemento del menú desplegable -->
                            <li><a href="#" class="dropdown-item">Opción 2</a></li> <!-- Segundo elemento del menú desplegable -->
                            <li><a href="#" class="dropdown-item">Opción 3</a></li> <!-- Tercer elemento del menú desplegable -->
                        </ul>
                    </li>
                </ul>
                <form action="" class="d-flex"> <!-- Formulario para la barra de búsqueda -->
                    <input type="text" class="form-control" placeholder="Buscar..."> <!-- Campo de texto para ingresar lo que se desea buscar -->
                </form>
            </div>
        </div>
    </div>
