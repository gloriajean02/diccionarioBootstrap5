Contenedores en Bootstrap 5

Comparativa


Container (máximo 1140px)


Container SM (mín. 576px)

Container

El contenedor básico, .container, tiene un ancho máximo fijo dependiendo del tamaño de la pantalla, lo que ayuda a alinear el contenido y mantenerlo centrado.


<div class="container bg-primary">
    <h3>Container</h3>
</div>


Container SM

El contenedor .container-sm está diseñado para ajustarse a dispositivos pequeños (pantallas mayores de 576px). Es útil para dispositivos como teléfonos móviles en modo horizontal o pantallas de baja resolución. 


<div class="container-sm bg-primary-subtle">
    <h3>Container SM</h3>
</div>


Container MD

El contenedor .container-md funciona igual que .container, pero está diseñado para ajustarse específicamente a dispositivos medianos en adelante (pantallas mayores de 768px).


<div class="container-md bg-danger">
    <h3>Container MD</h3>
</div>


Container LG

El contenedor .container-lg está diseñado para ajustarse a pantallas grandes (mayores de 992px). Se usa cuando quieres que el contenido ocupe más espacio en pantallas grandes.


<div class="container-lg bg-info">
    <h3>Container LG</h3>
</div>


Container XL

El contenedor .container-xl se ajusta a pantallas extra grandes (mayores de 1200px). Se utiliza cuando se desea más espacio para contenido en pantallas muy grandes, como en monitores de escritorio de alta resolución.


<div class="container-xl bg-warning">
    <h3>Container XL</h3>
</div>


Container XXL

El contenedor .container-xxl es para pantallas extremadamente grandes (mayores de 1400px). Es útil cuando se necesita un diseño totalmente expansivo para pantallas muy anchas.


<div class="container-xxl bg-success">
    <h3>Container XXL</h3>
</div>


Container FLUID

El contenedor .container-fluid se adapta al 100% del ancho de la pantalla sin restricciones, lo que permite que el contenido ocupe toda la pantalla independientemente de su tamaño.


<div class="container-fluid bg-success-subtle">
    <h3>Container FLUID</h3>
</div>
