Iconos en Bootstrap 5

Lo primero que debemos hacer es descargar el Source code (zip) aquí

Luego trasladamos toda la carpeta font dentro de nuestro proyecto, por ejemplo dentro de otra carpeta llamada bootstrap-icons, seleccionaremos el bootstrap-icons.min.css.

Después lo incluimos en el html así, incluyendo la segunda línea de código (que enlaza a la galería de iconos de bootstrap):

<link href="../bootstrap-icons/font/bootstrap-icons.min.css" rel="stylesheet">
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0-alpha1/dist/css/bootstrap.min.css">

Puedes consultar los códigos de los iconos aquí

El siguiente ícono es un círculo:



<i class="bi bi-0-circle-fill"></i>


El siguiente ícono es una batería, y hemos ajustado su tamaño a 50px:



<i class="bi bi-battery" style="font-size: 50px;"></i>


El siguiente ícono es un bug, y lo hemos coloreado de rojo:



<i class="bi bi-bug-fill" style="color: red;"></i>
