Flexbox en Bootstrap - Ejemplos básicos

1. Flex básico en padres

Sin Flex (div normal, ocupa toda la fila):

Elemento 1
Elemento 2
Elemento 3
Elemento 4
<div class="bg-success border p-3">Elemento 1</div>
<div class="bg-success border p-3">Elemento 2</div>
<div class="bg-success border p-3">Elemento 3</div>
<div class="bg-success border p-3">Elemento 4</div>

Con Flex (d-flex) (los elementos se ajustan al contenido horizontalmente):

Elemento 1
Elemento 2
Elemento 3
Elemento 4
<div class="d-flex">
    <div class="bg-success border p-3">Elemento 1</div>
    <div class="bg-success border p-3">Elemento 2</div>
    <div class="bg-success border p-3">Elemento 3</div>
    <div class="bg-success border p-3">Elemento 4</div>
</div>

4. Flex Grow, Shrink y Fill

Flex Grow (el elemento crece para ocupar el espacio disponible):

Grow 1
Normal
Normal

<div class="d-flex mb-4">
<div class="bg-primary border p-3 flex-grow-1">Grow 1</div>
<div class="bg-primary border p-3">Normal</div>
<div class="bg-primary border p-3">Normal</div>
</div>


Flex Shrink (el elemento se encoge si es necesario):

Caja grande
Shrink
Caja grande

<div class="d-flex mb-4">
<div class="bg-primary border p-3" style="width: 300px;">Caja grande</div>
<div class="bg-danger border p-3 flex-shrink-1" style="width: 300px;">Shrink</div>
<div class="bg-primary border p-3" style="width: 300px;">Caja grande</div>
</div>


Flex Fill (todos los elementos llenan el espacio disponible de manera igual):

Fill 1
Fill 2
Fill 3
<div class="d-flex mb-4">
<div class="bg-primary border p-3 flex-fill">Fill 1</div>
<div class="bg-primary border p-3 flex-fill">Fill 2</div>
<div class="bg-primary border p-3 flex-fill">Fill 3</div>
</div>

Start
